Instructions to compile sass file for css:

0) Install Ruby and Sass
	
	sudo gem install sass 

1) Compile sass file

	sass style.sass style.css

Project description:

The file index.html contains the main html of the project (the page is responsive, you can resize the window and mobile layout is displayed):
	
	- To search an artist or an album: write the name in search input box (search action is triggered after 850 ms) 
	- To display more artists or albums click on “SHOW ME MORE!” button
